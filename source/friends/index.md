欢迎加友链^_^
在本页留言，格式如下
```yml
- site: #网站的名字
  owner: #您的名字
  url: #您的网址
  desc: #简短描述
  image: #一张图片的url
  color: #方块颜色
```
示例

```yml
- site: cosの博客 # 站点名称cos
  url: https://ysx.cosine.ren/ # 站点网址
  owner: cos  #昵称
  desc: WA的一声就哭了 # 站点简介
  image: https://ysx.cosine.ren/img/avatar.jpg
  color: '#ffc0cb' # 图标色
```
